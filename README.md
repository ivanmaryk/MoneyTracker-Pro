💰 MoneyTracker Pro — финансовый учёт с категориями расходов
Интеллектуальный инструмент для учёта личных финансов с поддержкой категорий, бюджетов, статистики и экспорта данных.
Позволяет контролировать расходы, устанавливать лимиты на категории и анализировать траты в удобном формате.
Реализован на 7 языках программирования для демонстрации подходов к работе с данными и бюджетным учётом.

🧠 Концепция
MoneyTracker Pro — это система для управления личными финансами, которая позволяет:

✅ Добавлять расходы с указанием категории, суммы, даты и комментария.

✅ Управлять категориями — создавать, редактировать, удалять категории.

✅ Устанавливать бюджет на каждую категорию (месячный лимит).

✅ Просматривать статистику — общие траты, расходы по категориям, остаток по бюджету.

✅ Анализировать — процент использования бюджета, самые затратные категории.

✅ Экспортировать данные в CSV для анализа в Excel.

✅ Сохранять и загружать данные автоматически (JSON).

✅ Работать в интерактивном режиме с простыми командами.

🚀 Как запустить
Каждая версия самодостаточна. Установите зависимости (если есть) и запустите.

Python
bash
pip install matplotlib  # для графиков (опционально)
python finance_python.py
C++
bash
g++ -std=c++17 finance_cpp.cpp -o finance
./finance
Java
bash
javac finance_java.java && java finance_java
C# (.NET Core)
bash
dotnet run
Go
bash
go run finance_go.go
Rust
bash
cargo build --release && ./target/release/finance_rs
JavaScript (Node.js)
bash
node finance_js.js
